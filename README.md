# WeatherBoard

### Description
This application is able to search weather by city for 5 days forecast.
This app will store searching history.
Current day result will display:
- City,
- Date,
- Temperature
- Humidity
- Wind Speed
- UV index

Future day result will display:
- Date
- Temperature
- Humidity

### Tech
- HTML
- Javascript
- CSS
- API
